# Ideas & Proposals

This directory contains ideas, proposals, and feature requests for the tpcli tool and related workflows.

## Current Ideas

### [Entity Discovery and Instance-Agnostic Query Helpers](./entity-discovery-helpers.md)
Make TargetProcess queries work with entity names instead of hardcoded IDs, enabling scripts to be portable across instances and more user-friendly.

**Status:** Proposed
**Priority:** Medium
**Relates to:** PI Planning, Team Analysis, Automation

## How to Propose an Idea

1. Create a new markdown file in this directory
2. Use a clear, descriptive filename (e.g., `feature-name.md`)
3. Include:
   - **Problem Statement:** What's the issue you're trying to solve?
   - **Proposed Solution:** How should it work?
   - **Use Cases:** Real-world examples
   - **Technical Considerations:** Challenges and trade-offs
   - **Next Steps:** What needs to happen next?

4. Reference related ideas with links
5. Add your idea to this README

## Idea Lifecycle

- **Proposed:** Initial idea, gathering feedback
- **Approved:** Team consensus, ready for prototyping
- **In Progress:** Being implemented
- **Complete:** Shipped and documented
- **Deferred:** Decided not to pursue (for now)
